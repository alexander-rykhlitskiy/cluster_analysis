## About
Cluster analysis in pattern recognition.
## Requirements
Ruby 2.1.1
## Installation
```bash
gem install rmagick:2.13.3 gruff:0.5.1 pry
```
## Usage
```bash
ruby main.rb -i spoons.jpg -n 2

ruby main.rb --help
```
